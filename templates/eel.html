English Essence is a communicative English learning app. There are 100s of different types of quizzes so that anyone can practice them on the go along with 100s of video and audio lectures. There are also a group communication system which will allow anyone to keep practicing English with his/her peers. English Essence is the largest English Learning Platform in Bangladesh.

I joined this company in October 2020 as a Jr. software engineer when the company started its journey. At that time the company did not have any backend system & mobile application. And I had to build the entire infrastructure of the backend system in just 3 months from scratch.

My job was to build the whole backend system from the scratch on Django framework.

The main purpose of English Essence was to build a communicative english learning app. That app needs a backend system, so I had to make a backend system with 100+ API endpoint using Django Rest Framework.

I had to make a web frontend which is nothing but teacher's Dashboard where teacher's can design their course & upload necessary content for every course and lot of quizzes as well.

The system has the user types:
- Super User
- Teacher
- Student
Those three types user has different privilege.

Some of Mobile App Feature

1. Short lessons with audios and videos
2. Great teaching plan where anyone can learn English by just practicing.
3. More than 10 types of Quizzes types to help anyone learn English Faster.
4. A communicative group for the same level of English users, which promotes interaction between peers.

Backend system Development

- Built entire backend system from scratch. Designed database and algorithm for the advanced quiz system as well as communicative group. Design & Create more than 100 API endpoints for mobile app. Integrate Firebase for media storage & notification system, integrate celery & celery beat for background and period tasks. Integrate AWS S3 for DB backup, payment gateway adaptation, integrate Sendgrid for mail system.

- Design a dashboard with Teacher's & Admin user privilege where teacher's can upload all course materials and admin can review, verify, add new teacher, banned & delete users, send & set scheduler notification, take backup and many more.

- Use Docker for building project image & push to the Heroku container registry. Use PostgreSQL as database.

- Make all API endpoints documented using Swagger & OpenAPI. Also write manual documentation for all API endpoints in XL sheet. code comment & write entire development & production process in README.md file
