Cost and Time efficient Computer Aided Commercial Software for Blood Pathogens

This was a project I started back in January 2019 when I started to learn python and developed a project on a micro framework like flask. This was my Senior Design Project.

A web application which resides an AI Base Visual Recognition System. In which different AI based visual recognize system can be run behind the scene that detects red blood cell diseases.

An AI model has already been developed that detects Malaria with the accuracy of 97%. In the background, the microscopic image is then directly input to the system. As output every infected & non-infected cell are labeled as their respective diseases and provide a quantitative understanding of level of severity.

After getting the identity from admin a pathologist can sign up in this system and then will get access to the pathologist dashboard. Pathologists are responsible for generating the blood images from blood via microscope. And they can Add/View/Edit/Delete patient records as well.

After completing a test of a patient system will generate a report & save it to the database for future purpose.

I worked on this system for 1 years although my course was finished within 8 months.
