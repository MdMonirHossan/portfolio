A Modern Online Store for Home Decoration

This was a project I started back in 2020 when I started to learn Django framework and this was my 2nd project on Django framework. I developed this project within one month and I really wanted to make a fully functional E-Commerce site something like this project. This was one of my personal project.

A web application which is a fully functional E-Commerce website with user and guest checkout capabilities. There are two types of checkout system one is for logged in user and another is for guest user. Checkout functionality also available with paypal payment system.

The main focus of this eCommerce project is to integrate guest checkout ability along with user accounts. Forcing user to create an account for purchasing something can affect sales significantly.

Authenticated User Vs Guest Checkout
Authenticated users & guests have the same process with slight differences. Both users have the ability to add product in wishlist but user with account can see previous purchases as well as pending product while guests will have items stored in cookies and will have the option to create account after a successful purchase.

Authenticated User Process
Add item to cart → Edit Order → Checkout View pending and previous orders + Order details

Guest Checkout Process
Add item to cart → Edit Order → Checkout Create Account to view Order

Apply Cupon
Registered users have cupon facility, if the user is registered & has a valid cupon then that user will get discount on total amount.

After completing checkout process an confirmation email will be sent to user email.
