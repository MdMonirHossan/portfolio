Basha khujchi - A Smart Home Rental System

This projects was one of my academic project, we planned to build this project on PHP we didn't able to complete this project. After that I started to work on this project personally, this was a project I started back in 2020 when I started to learn Django Framework for the first time.

A smart & hassle free web application for home rental system of bangladesh where tenants can looking for vacant houses and house owners can rent their houses without any headache.

There is a communication system in this project by which both tenants & house owners can communicate with each other via this system.

The system is the simplest solution to the rental system. It will reduce hassle & save time for both tenants and house owners.

Technologies:
  • - Python
  • - Django
  • - HTML/CSS
  • - Bootstrap
  • - Javascript
  • - JQuery
  • - Postgres
